# Scoring Methodology

## Overview

The AI Evaluation Platform evaluates AI-generated answers using a retrieval-based scoring approach. The evaluation combines retrieved reference context with optional reference answers to produce multiple quality metrics.

---

# Evaluation Workflow

User Question

↓

Retrieve Relevant Context

↓

Compare AI Response

↓

Calculate Scores

↓

Generate Overall Score

↓

Produce Final Verdict

---

# Evaluation Metrics

## 1. Relevance

Definition

Measures how closely the retrieved context matches the user's question.

Method

- Keyword overlap
- Text similarity

Score Range

0.0 – 1.0

Higher is better.

---

## 2. Accuracy

Definition

Measures how closely the AI response matches the reference answer.

If no reference answer is provided, the retrieved context is used.

Method

- String similarity
- Keyword overlap

Score Range

0.0 – 1.0

Higher is better.

---

## 3. Hallucination

Definition

Measures unsupported information generated by the AI.

Method

Hallucination is estimated using the overlap between the AI response and retrieved context.

Lower hallucination indicates better factual grounding.

Score Range

0.0 – 1.0

Lower is better.

---

## 4. Completeness

Definition

Measures whether the AI response sufficiently covers the expected answer.

Method

Length comparison between the AI response and the reference answer (or retrieved context).

Score Range

0.0 – 1.0

Higher is better.

---

# Overall Score

The overall score is calculated as a weighted combination of the individual metrics.

Weights

- Relevance : 20%
- Accuracy : 45%
- Hallucination : 25%
- Completeness : 10%

Overall Score =

0.20 × Relevance

+

0.45 × Accuracy

+

0.25 × (1 − Hallucination)

+

0.10 × Completeness

---

# Verdict

| Overall Score | Verdict |
|---------------|---------|
| 0.90 – 1.00 | Excellent |
| 0.75 – 0.89 | Good |
| 0.55 – 0.74 | Average |
| Below 0.55 | Poor |

---

# Current Limitations

- Uses lexical similarity instead of deep semantic similarity.
- Retrieval quality depends on the indexed knowledge base.
- Hallucination detection is heuristic-based.
- Scores may not fully capture paraphrased answers.
